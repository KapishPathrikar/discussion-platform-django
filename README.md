# DiscX – Discussion Platform for Coders

A full-stack web application built using Django that allows users to create discussion rooms, collaborate on topics, and communicate through real-time messaging.

---

## 🚀 Features

- User Authentication (Login, Registration, Profile Management)
- Create, Join, and Manage Topic-Based Discussion Rooms
- Real-Time Messaging with Multi-line Support
- Search Functionality (by room name, topic, description)
- Topic-Based Room Categorization
- User Profiles with Avatar Support
- Responsive UI (Desktop + Mobile)

---

## 🛠️ Tech Stack

- **Backend:** Django (MVT Architecture), Python  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite  
- **Tools:** Git, GitHub
